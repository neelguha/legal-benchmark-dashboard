About

Code for replicating these results can be found here.

  • LegalBench — we use 159 tasks from the original LegalBench paper. We downsample each task to 50 samples.
  • BarExam — a publicly released subset of MBE questions, taken from Zheng et al. (2025).
  • HousingQA — a set of statutory reasoning questions in which the LLM answers a question with reference to a statute. These are taken from Zheng et al. (2025). We subsample the original dataset down, taking 50 per question type.
  • LEXam — English-language questions in the 32-choice regime from the original LEXam paper.
  • Hallucinations — a subset of questions from Dahl et al. (2024). We focus on four tasks with verifiable answers (affirm_reverse, case_existence, citation_retrieval, year_overruled), and subsample 100 from each task.

For reproducibility, we make the explicit benchmark data used to produce these results available here.

All models are run on their highest reasoning setting. Models are prompted to include their answers in a structured format.
